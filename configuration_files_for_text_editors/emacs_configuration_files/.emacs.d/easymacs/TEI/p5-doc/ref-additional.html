TEI P5: Guidelines for Electronic Text Encoding and Interchange

<additional>

additionalgroups additional information, combining bibliographic information about a manuscript, or surrogate copies of it with curatorial or administrative information.
Attributes: Global attributes only
Declaration
element additional
{
   att.global.attributes,
   ( adminInfo?, surrogates?, listBibl? )
}
Example
 <additional>
  <adminInfo>
   <recordHist>
<!-- record history here -->
   </recordHist>
   <custodialHist>
<!-- custodial history here -->
   </custodialHist>
  </adminInfo>
  <surrogates>
<!-- information about surrogates here -->
  </surrogates>
  <listBibl>
<!-- full bibliography here -->
  </listBibl>
 </additional>
