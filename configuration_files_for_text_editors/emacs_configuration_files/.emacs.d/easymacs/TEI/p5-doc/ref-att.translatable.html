att.translatable

att.translatableprovides attributes used to indicate the status of a translatable portion of an ODD document.
Attributes
In addition to global attributes
versionspecifies the version name or number of the source from which the translated version was derived
Status: Optional
Datatype data.word

The version may be a number, a letter, or a date
