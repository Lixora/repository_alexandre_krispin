<attDef>

attDef(attribute definition) contains the definition of a single attribute.
Classatt.identified
Declaration
element attDef
{
   att.global.attributes,
   att.identified.attributes,
   attribute usage { "req" | "mwa" | "rec" | "rwa" | "opt" }?,
   attribute ns { data.namespace }?,
   (
      macro.glossSeq,
      datatype?,
      defaultVal?,
      ( valList | valDesc )?,
      exemplum*,
      remarks*
   )
}
Attributes
[att.identified ]
usagespecifies the optionality of an attribute or element.
Status: Optional
Default: opt
Legal values are:
req(required)
mwa(mandatory when applicable )
rec(recommended )
rwa(recommended when applicable )
opt(optional )
ns(namespace) specifies the namespace to which this attribute belongs
Status: Optional
Datatype data.namespace
Default: http://www.tei-c.org/ns/1.0
Example
 <attDef usage="rec" ident="type">
  <desc>specifies a name conventionally used for this level of subdivision, e.g. <q>act</q>, <q>volume</q>,
   <q>book</q>, <q>section</q>, <q>canto</q>, etc.</desc>
 </attDef>
