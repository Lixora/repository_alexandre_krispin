TEI P5: Guidelines for Electronic Text Encoding and Interchange

<author>

authorin a bibliographic reference, contains the name of the author(s), personal or corporate, of a work; the primary statement of responsibility for any bibliographic item.
Attributes: Global attributes only
Classmodel.respLike
Declaration
element author { att.global.attributes, macro.phraseSeq }
Example
 <author>British Broadcasting Corporation</author>
 <author>La Fayette, Marie Madeleine Pioche de la Vergne, comtesse de (1634–1693)</author>
Note

Particularly where cataloguing is likely to be based on the content of the header, it is advisable to use generally recognized authority lists for the exact form of personal names.

In the case of a broadcast, use this element for the name of the company or network which broadcasts the program.
