TEI P5: Guidelines for Electronic Text Encoding and Interchange

<biblFull>

biblFullcontains a fully-structured bibliographic citation, in which all components of the TEI file description are present.
Attributes: [att.declarable ]
Classmodel.biblLike att.declarable
Declaration
element biblFull
{
   att.global.attributes,
   att.declarable.attributes,
   ( macro.fileDescPart, sourceDesc* )
}
Example
 <biblFull>
  <titleStmt>
   <title>The Feminist Companion to Literature in English: women writers from the middle ages to the present</title>
   <author>Blain, Virginia</author>
   <author>Clements, Patricia</author>
   <author>Grundy, Isobel</author>
  </titleStmt>
  <editionStmt>
   <edition>UK edition</edition>
  </editionStmt>
  <extent>1231 pp</extent>
  <publicationStmt>
   <publisher>Yale University Press</publisher>
   <pubPlace>New Haven and London</pubPlace>
   <date>1990</date>
  </publicationStmt>
  <sourceDesc>
   <p>No source: this is an original work</p>
  </sourceDesc>
 </biblFull>
