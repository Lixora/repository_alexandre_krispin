TEI P5: Guidelines for Electronic Text Encoding and Interchange

<biblItem>

biblItem(bibliographic citation) contains a bibliographic citation in which only bibliographic subelements appear.
Classmodel.biblLike att.declarable att.typed
Declaration
element biblItem
{
   att.global.attributes,
   att.declarable.attributes,
   att.typed.attributes,
   attribute form { data.word }?,
   attribute status { text }?,
   attribute role { text }?,
   (
      title
    | biblItem
    | imprint
    | model.nameLike
    | model.biblPart
    | model.dateLike
   )+
}
Attributes
[att.declarable att.typed ]
formindicates the physical form of the object of this bibliographic entry.
Status: Optional
Datatype data.word
Sample values include:
book.hardcover
book.paperback
video.VHS
video.DVD
audio.casette
audio.CD
statusindicates the publication status of the object of this bibliographic entry.
Status: Optional
Sample values include:
manuscript(the item is a manuscript)
forthcoming(the item is being published, but has not been released yet)
inPrint(the item is currently being published)
roleindicates the relationship of this bibliographic entry to that of its parent bibl or biblItem.
Status: Optional
Sample values include:
host(this bibliographic item has a host relationship to the item encoded in the parent bibl or biblItem; e.g., this item is the magazine in which the article described by the parent bibl or biblItem appeared.)
original(the item is original source)
workReviewed(this bibliographic item was reviewed in the ancestor work)
edition(indicates that this entry contains separate information for a particular edition of the parent bibl or biblItem)
basedOn(indicates that the parent bibl or biblItem is the base work from which this one was derived)
referencedBy(indicates that the element is a reference for the work in question)
copyOf(indicates that this entry is for a facsimile of that which is referred to by the parent bibl or biblItem)
Example
 <biblItem>
  <author>Blain, Virginia</author>
  <author>Clements, Patricia</author>
  <author>Grundy, Isobel</author>
  <title level="m">The Feminist Companion to Literature in English</title>
  <title level="m" type="subordinate">Women Writers from the Middle Ages to the Present</title>
  <edition>first edition</edition>
  <imprint>
   <publisher>Yale University Press</publisher>
   <pubPlace>New Haven and London</pubPlace>
   <date>1990</date>
  </imprint>
 </biblItem>
Example
 <biblItem>
  <title level="a">Changes in physical examination caused by use of spinal immobilization</title>
  <author>March, Juan A. MD</author>
  <author>Ausband, Stephen C. MD</author>
  <author>Brown, Lawrence H. EMT-P</author>
  <biblScope type="pages">421–424</biblScope>
  <biblItem role="host">
   <title level="j">Prehospital Emergency Care</title>
   <biblScope type="volume">6</biblScope>
   <biblScope type="issue">4</biblScope>
   <date>October/December 2002</date>
   <publisher>Hanley & Belfus</publisher>
   <editor>Menegazzi, James J. PhD</editor>
  </biblItem>
 </biblItem>
