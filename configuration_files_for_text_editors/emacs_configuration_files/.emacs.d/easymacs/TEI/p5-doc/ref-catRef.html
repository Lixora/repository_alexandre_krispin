TEI P5: Guidelines for Electronic Text Encoding and Interchange

<catRef>

catRef(category reference) specifies one or more defined categories within some taxonomy or text typology.
Declaration
element catRef
{
   att.global.attributes,
   attribute target { list { data.pointer+ } },
   attribute scheme { data.pointer }?,
   empty
}
Attributes
In addition to global attributes
targetidentifies the categories concerned
Status: Required
Datatype [n <= unbounded] data.pointer
Values: A series of one or more space-separated pointers (URIs) to <category> elements, typically in the header of the current document
schemeidentifies the classification scheme within which the set of categories concerned is defined
Status: Optional
Datatype data.pointer
Values: identifier of the associated <taxonomy> element.
Example
 <catRef target="#ba #c1" scheme="#s1"/>
Note

The scheme attribute need be supplied only if more than one taxonomy has been declared
