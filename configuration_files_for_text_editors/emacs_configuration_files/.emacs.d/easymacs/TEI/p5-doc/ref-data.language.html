TEI P5: Guidelines for Electronic Text Encoding and Interchange

data.language

data.languagedefines the range of attribute values used to identify a particular combination of human language and writing system
Declaration
data.language = xsd:language
Note
The values for this attribute are language ‘tags’ as defined in RFC 3066 or its successor. Examples include
sn
Shona
zh-TW
Taiwanese
en-SL
English as spoken in Sierra Leone
pl
Polish
es-MX
Spanish as spoken in Mexico
