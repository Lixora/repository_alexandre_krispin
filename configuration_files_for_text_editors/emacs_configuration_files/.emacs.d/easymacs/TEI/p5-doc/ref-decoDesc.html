<decoDesc>

decoDesccontains a description of the decoration of a manuscript, either as a sequence of paragraphs, or as a sequence of topically organised decoNote elements.
Attributes: Global attributes only
Classmodel.physDescPart
Declaration
element decoDesc { att.global.attributes, ( model.pLike+ | decoNote+ ) }
Example
 <decoDesc>
  <p>The start of each book of the Bible with a 10-line historiated illuminated initial; prefaces decorated with 6-line blue initials with red penwork flourishing; chapters marked by 3-line plain red initials; verses with 1-line initials, alternately blue or red.</p>
 </decoDesc>
