TEI P5: Guidelines for Electronic Text Encoding and Interchange

<distance>

distance(distance) that part of a relative temporal or spatial expression which indicates the distance between the place or time denoted by it and the place or time referred to within it.
Classmodel.datePart att.datePart model.placeNamePart att.typed
Declaration
element distance
{
   att.global.attributes,
   att.datePart.attributes,
   att.typed.attributes,
   attribute exact { "true" | "false" | "unknown" }?,
   macro.phraseSeq
}
Attributes
[att.datePart att.typed ]
exactindicates the degree of accuracy associated with the distance.
Status: Mandatory when applicable
Datatype data.enumerated
Default: unknown
Legal values are:
true(The distance is exact.)
false(The distance is approximate.)
unknown(Accuracy unavailable or unknown.)
Example
 <placeName>
  <distance>two miles</distance>
  <offset>north east of</offset>
  <placeName>Manchester</placeName>
 </placeName>
Example
 <date>
  <distance dur="P14D">a fortnight</distance>
  <offset>after</offset>
  <date type="occasion">Michaelmas</date>
 </date>
Example
 <time>
  <distance>20 minutes</distance>
  <offset>before</offset>
  <time type="descriptive">noon</time>.
  </time>
Note

This is a specialized form of measure, used only within relative temporal and spatial expressions.
