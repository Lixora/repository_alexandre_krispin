TEI P5: Guidelines for Electronic Text Encoding and Interchange

<div1>

div1(level-1 text division) contains a first-level subdivision of the front, body, or back of a text.
Attributes: [att.divLike att.declaring ]
Classatt.divLike att.declaring model.div1Like
Declaration
element div1
{
   att.global.attributes,
   att.divLike.attributes,
   att.declaring.attributes,
   (
      ( model.divWrapper | model.global )*,
      (
         ( ( model.div2Like | model.divGenLike ), model.global* )+
       | (
            ( macro.component, model.global* )+,
            ( ( model.div2Like | model.divGenLike ), model.global* )*
         )
      ),
      ( ( model.divWrapper | model.divWrapper.bottom ), model.global* )*
   )
}
Example
 <div1 xml:id="levi" n="I" type="part">
  <head>Part I: Of Man </head>
  <div2 xml:id="levi1" n="1" type="chapter">
   <head>Chap. I. Of Sense </head>
   <p>Concerning the Thoughts of man... </p>
  </div2>
 </div1>
 <div1 xml:id="levii" n="II" type="part">
  <head>Part II: Of Common-Wealth</head>
 </div1>
Note

any sequence of low-level structural elements, possibly grouped into lower subdivisions.
