<f>

f(Feature) represents a feature value specification, that is, the association of a name with a value of any of several different types.
Declaration
element f
{
   att.global.attributes,
   attribute name { data.name },
   attribute fVal { data.pointer }?,
   model.featureVal*
}
Attributes
In addition to global attributes
nameprovides a name for the feature.
Status: Required
Datatype data.name
Values: Any name.
fValreferences any element which can be used to represent the value of a feature.
Status: Optional
Datatype data.pointer
Values: the identifier of an element representing a feature value

If this attribute is supplied as well as content, the value referenced is to be unified with that contained.

Example
 <f name="gender">
  <symbol value="feminine"/>
 </f>
Note

If the element is empty then a value must be supplied for the fVal attribute.
