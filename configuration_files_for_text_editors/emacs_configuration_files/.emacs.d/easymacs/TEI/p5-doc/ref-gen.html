TEI P5: Guidelines for Electronic Text Encoding and Interchange

<gen>

gen(gender) identifies the morphological gender of a lexical item, as given in the dictionary.
Attributes: [att.lexicographic ]
Classmodel.entryParts model.morphLike att.lexicographic
Declaration
element gen
{
   att.global.attributes,
   att.lexicographic.attributes,
   macro.paraContent
}
Example
 <entry>
  <form>
   <orth>pamplemousse</orth>
  </form>
  <gramGrp>
   <pos>noun</pos>
   <gen>masculine</gen>
  </gramGrp>
 </entry>
Note

May contain character data and phrase-level elements. Typical content will be masculine, feminine, neuter etc.

This element is synonymous with <gram type="gender">.
