TEI P5: Guidelines for Electronic Text Encoding and Interchange

<gramGrp>

gramGrp(grammatical information group) groups morpho-syntactic information about a lexical item, e.g. pos, gen, number, case, or iType (inflectional class).
Attributes: [att.lexicographic ]
Classmodel.entryParts.top model.entryParts model.gramPart att.lexicographic
Declaration
element gramGrp
{
   att.global.attributes,
   att.lexicographic.attributes,
   (
      text
    | model.gLike
    | model.phrase
    | model.inter
    | model.gramPart
    | model.global
   )*
}
Example
 <entry>
  <form>
   <orth>luire</orth>
  </form>
  <gramGrp>
   <pos>verb</pos>
   <subc>intransitive</subc>
  </gramGrp>
 </entry>
Note

[Obviously wrong!] May contain character data mixed with any other elements defined in the dictionary tag set.
