<handDesc>

handDesccontains a description of all the different kinds of writing used in a manuscript.
Classmodel.physDescPart
Declaration
element handDesc
{
   att.global.attributes,
   attribute hands { data.count }?,
   ( handNote | model.pLike )+
}
Attributes
In addition to global attributes
handsspecifies the number of distinct hands identified within the manuscript
Status: Optional
Datatype data.count
Values: A whole number, or the string "many"
Example
 <handDesc>
  <p>Written in <term>angelicana formata</term>.</p>
 </handDesc>
Example
 <handDesc hands="2">
  <p>The manuscript is written in two contemporary hands, otherwise unknown, but clearly those of practised scribes. Hand I writes ff. 1r-22v and hand II ff. 23 and 24. Some scholars, notably Verner Dahlerup and Hreinn Benediktsson, have argued for a third hand on f. 24, but the evidence for this is insubstantial.</p>
 </handDesc>
