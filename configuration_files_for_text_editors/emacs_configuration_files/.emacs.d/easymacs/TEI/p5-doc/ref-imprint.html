TEI P5: Guidelines for Electronic Text Encoding and Interchange

<imprint>

imprintgroups information relating to the publication or distribution of a bibliographic item.
Attributes: Global attributes only
Declaration
element imprint
{
   att.global.attributes,
   ( ( ( model.imprintPart ) | ( model.dateLike ) ), model.global* )+
}
Example
 <imprint>
  <pubPlace>Oxford</pubPlace>
  <publisher>Clarendon Press</publisher>
  <date>1987</date>
 </imprint>
