TEI P5: Guidelines for Electronic Text Encoding and Interchange

<listBibl>

listBibl(citation list) contains a list of bibliographic citations of any kind.
Attributes: [att.declarable ]
Classmodel.listLike att.declarable model.msItemPart
Declaration
element listBibl
{
   att.global.attributes,
   att.declarable.attributes,
   (
      ( ( model.divWrapper ) | ( model.global ) )*,
      ( ( model.biblLike ), model.global* )+,
      ( ( model.divWrapper.bottom ), model.global* )*
   )
}
Example
 <listBibl>
  <head>Works consulted</head>
  <bibl>Blain, Clements and Grundy: Feminist Companion to Literature in English (Yale, 1990) </bibl>
  <biblStruct>
   <analytic>
    <title>The Interesting story of the Children in the Wood</title>
   </analytic>
   <monogr>
    <title>The Penny Histories</title>
    <author>Victor E Neuberg</author>
    <imprint>
     <publisher>OUP</publisher>
     <date>1968</date>
    </imprint>
   </monogr>
  </biblStruct>
 </listBibl>
