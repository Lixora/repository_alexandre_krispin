TEI P5: Guidelines for Electronic Text Encoding and Interchange

<listPerson>

listPersoncontains a list of descriptions, each of which provides information about an identifiable person or a group of people, for example the participants in a language interaction, or the people referred to in a historical source.
Attributes: [att.typed att.declarable ]
Classatt.typed att.declarable model.biblLike model.personLike
Declaration
element listPerson
{
   att.global.attributes,
   att.typed.attributes,
   att.declarable.attributes,
   ( model.pLike+ | ( ( person | personGrp )+, particLinks? ) )
}
Example
 <listPerson type="respondents">
  <personGrp xml:id="PXXX"/>
  <person xml:id="P1234" sex="2" age="42"/>
  <person xml:id="P4332" sex="1" age="43"/>
  <particLinks>
   <relation type="personal" name="spouse" mutual="#P1234 #P4332"/>
  </particLinks>
 </listPerson>
Note

The type attribute may be used to distinguish lists of people of a particular type if convenient.
