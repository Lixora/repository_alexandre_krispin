<msItemStruct>

msItemStructcontains a structured description for an individual work or item within the intellectual content of a manuscript or manuscript part.
Classmodel.msItemPart att.msExcerpt
Declaration
element msItemStruct
{
   att.global.attributes,
   att.msExcerpt.attributes,
   attribute class { data.code }?,
   (
      locus?,
      (
         model.pLike+
       | (
            author*,
            respStmt*,
            title*,
            rubric?,
            incipit?,
            msItemStruct*,
            explicit?,
            finalRubric?,
            colophon*,
            decoNote*,
            listBibl*,
            bibl*,
            filiation*,
            model.noteLike*,
            textLang?
         )
      )
   )
}
Attributes
[att.msExcerpt ]
classidentifies the text types or classifications applicable to this item
Status: Optional
Datatype data.code
Values: One or more codes, each of which is used as the identifier for a text classification element supplied in the TEI Header <textClass> element.
