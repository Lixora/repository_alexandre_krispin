<namespace>

namespacesupplies the formal name of the namespace to which the elements documented by its children belong.
Declaration
element namespace
{
   att.global.attributes,
   attribute name { data.namespace },
   tagUsage+
}
Attributes
In addition to global attributes
namethe full formal name of the namespace concerned.
Status: Required
Datatype data.namespace
Example
 <namespace name="http://www.tei-c.org/ns/1.0">
  <tagUsage
    gi="hi"
    occurs="28"
    withId="2"
    render="#it"> Used only to mark English words italicised in the copy text </tagUsage>
 </namespace>
