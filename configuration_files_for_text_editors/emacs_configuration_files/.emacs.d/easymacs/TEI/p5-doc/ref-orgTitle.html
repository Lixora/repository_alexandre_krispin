TEI P5: Guidelines for Electronic Text Encoding and Interchange

<orgTitle>

orgTitle(organization title) contains the proper name component of an organizational name.
Declaration
element orgTitle
{
   att.global.attributes,
   attribute type { data.enumerated }?,
   attribute reg { text }?,
   macro.phraseSeq
}
Attributes
In addition to global attributes
typemore fully describes the organization title. Possible values include formal, colloquial, acronym, etc.
Status: Mandatory when applicable
Datatype data.enumerated
Values: any string of characters
reggives a normalized or regularized form of the organization title.
Status: Optional
Datatype text
Values: Any string of characters.

In providing a ‘regularized’ form, no claim is made that the form in the source text is incorrect; the regularized form is simply that chosen as the main form for purposes of unifying variant forms under a single heading.

Example
Mr Frost will be able to earn an extra fee from  <orgName type="media" key="BSB1">
  <orgTitle type="acronym">BSkyB</orgTitle>
 </orgName>
rather than the
 <orgName type="media" key="BBC1">
  <orgTitle type="acronym" reg="British Broadcasting Corporation">BBC</orgTitle>
 </orgName>
