TEI P5: Guidelines for Electronic Text Encoding and Interchange

<orth>

orth(orthographic form) gives the orthographic form of a dictionary headword.
Classmodel.entryParts model.formPart att.lexicographic
Declaration
element orth
{
   att.global.attributes,
   att.lexicographic.attributes,
   attribute type { data.enumerated }?,
   attribute extent { data.enumerated }?,
   macro.paraContent
}
Attributes
[att.lexicographic ]
typegives the type of spelling.
Status: Optional
Datatype data.enumerated
Values: Any convenient word or phrase, e.g. lat (latinate),std (standard), trans (transliterated), etc.

extentgives the extent of the orthographic information provided.
Status: Optional
Datatype data.enumerated
Default: full
Sample values include:
full(full form)
pref(prefix)
suff(suffix)
part(partial)
