<persEvent>

persEventcontains a description of a particular event of significance in the life of a person.
Attributes: [att.datable att.editLike att.typed att.naming ]
Classatt.datable att.editLike att.typed att.naming model.persEventLike
Declaration
element persEvent
{
   att.global.attributes,
   att.datable.attributes,
   att.editLike.attributes,
   att.typed.attributes,
   att.naming.attributes,
   (
      label?,
      model.dateLike?,
      placeName?,
      model.pLike*,
      ( model.noteLike | model.biblLike )*
   )
}
Example
 <persEvent key="#MAT">
  <label>matriculation</label>
  <date value="1972-10-12">12 Oct 1972</date>
 </persEvent>
