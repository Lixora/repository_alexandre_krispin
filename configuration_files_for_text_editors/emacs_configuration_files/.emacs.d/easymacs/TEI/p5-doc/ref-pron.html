TEI P5: Guidelines for Electronic Text Encoding and Interchange

<pron>

pron(pronunciation) contains the pronunciation(s) of the word.
Classmodel.entryParts model.formPart att.lexicographic
Declaration
element pron
{
   att.global.attributes,
   att.lexicographic.attributes,
   attribute extent { data.enumerated }?,
   attribute notation { data.enumerated }?,
   macro.paraContent
}
Attributes
[att.lexicographic ]
extentindicates whether the pronunciation is for whole word or part.
Status: Optional
Datatype data.enumerated
Default: full
Sample values include:
full(full form)
pref(prefix)
suff(suffix)
part(partial)

notationindicates what notation is used for the pronunciation, if more than one occurs in the machine-readable dictionary.
Status: Required when applicable
Datatype data.enumerated
Values: Sample values: IPA, Murray, ...

Example
 <entry>
  <form>
   <orth>obverse</orth>
   <pron>&ACCENTäb-&accentvərs</pron>, <pron extent="prefix">äb-&ACCENT</pron>, <pron extent="prefix">əb-&ACCENT</pron>
  </form>
  <gramGrp>
   <pos>n</pos>
  </gramGrp>
 </entry>
