TEI P5: Guidelines for Electronic Text Encoding and Interchange

<recordHist>

recordHistprovides information about the source and revision status of the parent manuscript description itself.
Attributes: Global attributes only
Declaration
element recordHist
{
   att.global.attributes,
   ( model.pLike+ | ( source, change* ) )
}
Example
 <recordHist>
  <source>
   <p>Derived from <ref target="#IMEV123">IMEV 123</ref> with additional research
    by P.M.W.Robinson</p>
  </source>
  <change date="1999-06-23">
   <name>LDB</name> (editor)
   checked examples against DTD version 3.6
   </change>
 </recordHist>
