TEI P5: Guidelines for Electronic Text Encoding and Interchange

<remarks>

remarkscontains any commentary or discussion about the usage of an element, attribute, class, or entity not otherwise documented within the containing element.
Attributes: Global attributes only
Declaration
element remarks { att.global.attributes, macro.componentSeq }
Example
 <remarks>
  <p>This element is probably redundant.</p>
 </remarks>
Note

Contains at least one paragraph, unless it is empty.

As defined in ODD, must contain paragraphs; should be special.para
