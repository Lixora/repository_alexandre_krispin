TEI P5: Guidelines for Electronic Text Encoding and Interchange

<revisionDesc>

revisionDesc(revision description) summarizes the revision history for a file.
Attributes: Global attributes only
Declaration
element revisionDesc { att.global.attributes, ( list | change+ ) }
Example
 <revisionDesc>
  <change date="1991-11-11" who="#EMB"> deleted chapter 10 </change>
 </revisionDesc>
Example
 <revisionDesc>
  <change date="1991-11-11" who="#EMB">Chapitre 10 supprimé</change>
 </revisionDesc>
Note

Record changes with most recent changes at the top of the list.
