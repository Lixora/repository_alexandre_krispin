<spanGrp>

spanGrp(span group) collects together span tags.
Attributes: [att.interpLike ]
Classatt.interpLike model.global.meta
Declaration
element spanGrp { att.global.attributes, att.interpLike.attributes, span* }
Example
 <spanGrp resp="#TMA" type="collection of structural units">
  <span type="introduction" from="#s1" to="#s3"/>
  <span type="conflict" from="#s4a"/>
  <span type="climax" from="#s4b"/>
  <span type="revenge" from="#s5" to="#sx"/>
  <span type="reconciliation" from="#nil1"/>
  <span type="aftermath" from="#p2" to="#p4"/>
 </spanGrp>
Note

Any number of <span> elements.
