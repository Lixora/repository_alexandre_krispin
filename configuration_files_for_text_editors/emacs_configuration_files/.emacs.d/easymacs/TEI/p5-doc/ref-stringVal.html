TEI P5: Guidelines for Electronic Text Encoding and Interchange

<stringVal>

stringValcontains the intended expansion for the entity documented by an patternSpec element, enclosed by quotation marks.
Attributes: Global attributes only
Declaration
element stringVal { att.global.attributes, text }
Example
 <stringVal>"the choice of quotes isn't always unimportant"</stringVal>
Example

System entities should include the SYSTEM keyword within the content of this element, as shown:

 <stringVal>SYSTEM 'teiclasses"'</stringVal>
Note

The content of this element is the replacement text for the named entity, including any keywords, and surrounded by appropriate quotation marks.
