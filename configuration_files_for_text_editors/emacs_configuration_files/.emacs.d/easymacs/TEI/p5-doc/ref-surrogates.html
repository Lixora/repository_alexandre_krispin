TEI P5: Guidelines for Electronic Text Encoding and Interchange

<surrogates>

surrogatescontains information about any digital or photographic representations of the manuscript being described which may exist in the holding institution or elsewhere.
Attributes: Global attributes only
Declaration
element surrogates { att.global.attributes, macro.specialPara }
Example
 <surrogates>
  <p>
   <bibl>
    <title type="gmd">diapositive</title>
    <idno>AM 74 a, fol.</idno>
    <date>May 1984</date>
   </bibl>
   <bibl>
    <title type="gmd">b/w prints</title>
    <idno>AM 75 a, fol.</idno>
    <date>1972</date>
   </bibl>
  </p>
 </surrogates>
