TEI P5: Guidelines for Electronic Text Encoding and Interchange

<textLang>

textLangdescribes the languages and writing systems used by a manuscript (as opposed to its description, which is described in the langUsage element)
Classmodel.msItemPart
Declaration
element textLang
{
   att.global.attributes,
   attribute mainLang { data.language }?,
   attribute otherLangs { list { data.language* } }?,
   macro.phraseSeq
}
Attributes
In addition to global attributes
mainLangsupplies a code which identifies the chief language used in the manuscript.
Status: Optional
Datatype data.language
Values: a recognised language code using ISO 3066 or succesor, which may additionally be documented by a <language> element in the header
otherLangsone or more codes identifying any other languages used in the manuscript.
Status: Optional
Datatype [n >= 0] [n <= unbounded] data.language
Values: a list of codes, each of which is a recognised language code using RFC 3066 or succesor, which may additionally be documented by a <language> element in the header
Example
 <textLang mainLang="en" otherLangs="la"> Predominantly in English with Latin glosses</textLang>
