<valItem>

valItem(value definition) contains a single value and gloss pair for an attribute.
Attributes: [att.identified ]
Classatt.identified
Declaration
element valItem
{
   att.global.attributes,
   att.identified.attributes,
   ( macro.glossSeq )
}
Example
 <valItem ident="dub">
  <altIdent xml:lang="fr">dou</altIdent>
  <equiv name="unknown"/>
  <gloss>dubious</gloss>
 </valItem>
