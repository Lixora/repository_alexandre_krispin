Bookmark Operations (F2)

Key Command Description



F2 Next Bookmark Go to the next bookmark in the file

Shft-F2 Previous Bookmark Go to the previous bookmark in the file

Alt-F2 Toggle Bookmark Set or unset bookmark at current line

Ctrl-F2 List Bookmarks Display a menu of previously named bookmarks

Shft-Ctrl-F2 Name Bookmark Set a named bookmark at the current position

(hit “q” to quit help, Alt-Shift-F1 to return to the main help page or “l” to backtrack)
