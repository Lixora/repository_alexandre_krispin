Commands for Specific File-types (F10-F12)

With the exception of unmodified F10, function keys F10-F12 are mostly reserved for functions that are specific to the particular type of file being edited. If nothing else is defined for a particular file-type, the default is that F12 compiles and F11 runs. Note also that for some types of files, Shft-F1 produces context-sensitive help.

F10 on its own is the key that allows you to navigate the menus with the keyboard; but note that this does not yet work under GTK for GNU/Linux.

Here are the file-types for which there are special keys:

  • LaTeX files
  • XML files
  • Directory listings
