Commands for help and finding things (F1)

Key Command Description


F1 Help Show the main Easymacs help page

Shft-F1 Contextual Help Show help on the element near the cursor. Only some file-types implement this.

Alt-F1 Open File Open file, selecting it not with a dialog box, but with the native Emacs interface. At the prompt below, type the name of a file or directory. Use the Tab key to auto-complete file and directory names.

Shft-Alt-F1 Open Recent File Show a menu of recently opened files; hit return to open the selected file

Ctrl-F1 Insert File Like open file, but insert the contents of another file into the

Shft-Ctrl-F1 Locate File Locate a file on disk by typing its name. Requires the locate command and its database to be installed.

Ctrl-h Emacs Help Access the Emacs help system with Ctrl-h. Be aware that many keystrokes it describes are different in Easymacs, and it uses a different vocabulary to describe things.

(hit F1 or "l" to return to the main help page)
