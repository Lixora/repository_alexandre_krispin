Directory listings

Key Command Description


F12 Hide/Show Toggle hiding of “uninteresting” files in the current directory.

r Edit Filenames This is a powerful and potentially dangerous command that enables you to edit the names of files in a directory. Hit Ctrl-C twice when you are finished to write the changes to the names of the files on disk.

s Sort Hit “s”, and then the next key changes the sorting criterion: s for size, x for extension, t for time, n for name. Capitalize the letter to sort in reverse.
