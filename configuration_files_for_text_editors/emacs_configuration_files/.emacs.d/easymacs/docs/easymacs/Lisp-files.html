Lisp files

These are some keystrokes for Slime, the Emacs development environment for Common Lisp. You must install a Common Lisp implementation separately. Some of these keystrokes work for Emacs Lisp files as well.

You can move from one compiler note to the next with Alt-p (for previous) and Alt-n (for next).

Key Command Description



Shft-Return Fuzzy Complete Auto-complete the symbol at the cursor.

Ctrl-Shft-Ret Complete form Auto-complete the current form, e.g. function arglist.

Shft-F1 Lisp help Help on the symbol at the cursor from the draft Common Lisp standard for CL, and from the Emacs documentation for Elisp.

Alt-Q Indent defun Indent top-level form or comment.

F12 Eval defun Evaluate top-level form around cursor.

Shft-F12 Compile defun Compile top-level form around cursor.

Alt-F12 Eval file Evaluate contents of current file.

Alt-Shft-F12 Compile file Compile and load contents of current file.

Ctrl-F12 Expand-1 Macroexpand form at cursor.

Ctrl-Shft-F12 Expand-all Recursively macroexpand form at cursor.

F11 Eval last expr Evaluate the expression before the cursor.

Shft-F11 Eval expr Prompt for an expression to evaluate.

Alt-F11 Eval expr As F11, evaluate the expression before the cursor, but put the results in a buffer.

Alt-F11 Pretty expr As Alt-F11, but pretty-print the result of the evaluation before putting it in a buffer.

Ctrl-F11 Inspector Eval an expression and inspect the result.

Ctrl-Shft-F11 Describe Describe the symbol at the cursor.

F10 Switch Switch from a Lisp file to the REPL and back, via the debugger, if there is one active.

Shft-F10 Selector Select from a range of different Slime-related buffers.

Alt-F10 Check parens Check balancing of parens.

Alt-Shft-F10 Sync Set Lisp's package and directory to the values in current buffer.

Ctrl-F10 Show edits Toggle the mode in which Slime highlights code that has not yet been compiled.

Ctrl-Shft-F10 Trace Toggle tracing of function at the cursor.
